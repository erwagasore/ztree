# ztree

Format-agnostic document tree library for Zig.

## Quickstart

```bash
# Clone
git clone git@github.com:erwagasore/ztree.git
cd ztree

# Build
zig build

# Test
zig build test
```

### Usage

Add `ztree` as a dependency in your `build.zig.zon`, then import it:

```zig
const ztree = @import("ztree");
```
